start					开始调试,会停在main函数第一行语句前面等待

q						退出调试

l						列出源代码,每次十行,每次会接着上次往下显示.
l number				列出第number行开始的代码,但是在中间显示.
l 函数名				列出某个函数的源码(该函数必须存在当前文件中)

n						执行下一条语句,如果该语句使用了一个函数,会跳过该函数并直接操作.
s						执行下一条语句,如果该语句使用了一个函数,会进入该函数.

finish 					执行到对应函数返回,并在对应调用了该函数的语句停下来.

bt						查看函数栈顺序以及传入的参数的值.

info					info可以简化为i之后可以使用许多功能
i locals				查看该函数栈中所有局部变量的值
i b						查看所有断点信息

display value_name		跟踪一个变量,每次停下都显示他的值
undisplay				取消之前跟踪的变量


b 行号					在某行设置断点
b 函数名				把某个函数设置为断点,若该函数在文件中有定义,则在该函数停下后会进入该函数
delete breakpoints	n	删除NUM号为n的断点,不输入n时删除所有断点.

r						从头开始连续而非单步执行程序,遇到断点会停下.
c						从当前位置开始连续而非单步执行程序,遇到断点会停下.

disable breakpoints	n   禁用num号为n的断点,如果无n则会禁用所有断点
enable breakpoints	n   启用num号为n的断点,如果无n则会启用所有断点

set var 变量=value 		设置变量的值

p 变量名				打印变量的值

pt 变量名				查看该变量的类型
