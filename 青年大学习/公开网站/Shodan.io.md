**Shodan.io** 经常被称为“黑客界的谷歌”**。但与谷歌用来搜索网页（如文本、图片）不同，Shodan 是专门用来**搜索连接到互联网的各种硬件设备的搜索引擎。

简单来说，只要一个设备连上了互联网，并且暴露了公网 IP，Shodan 就能把它扫描出来。

### 1. 它能搜到什么？

Shodan 每天 24 小时都在对全球的 IP 地址进行全网扫描，抓取这些设备返回的“Banner”（服务指纹/横幅信息）。通过它，你可以搜到：

- **传统网络设备：** 服务器、路由器、交换机、网络打印机、防火墙。
- **物联网设备（IoT）：** 家用智能摄像头、智能冰箱、网络电视、红绿灯控制系统。
- **工业控制系统（ICS/SCADA）：** 甚至是水坝控制阀、发电厂、加油站油表、医院的医疗设备。
- **数据库与服务：** 暴露在公网且没有设密码的 Redis、MongoDB、Elasticsearch 数据库，或者各种开着 SSH、RDP 远程桌面的机器。

### 2. Shodan 是如何工作的？

普通搜索引擎（如 Google）是通过爬虫去读取网页上的 HTML 文本。 而 Shodan 是直接向各种 IP 地址的端口发送请求（比如握手请求），然后记录设备返回的响应信息。

这些响应信息里通常包含：

- 设备的操作系统（Linux, Windows 等）
- 运行的软件版本（例如 `Apache/2.4.41`）
- 设备的地理位置、所属运营商
- 是否存在已知的安全漏洞（CVE 编号）
