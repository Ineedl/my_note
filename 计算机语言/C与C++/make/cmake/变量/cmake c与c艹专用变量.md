## CMAKE_C_STANDARD_REQUIRED
设置必须指定C标准版本

默认为OFF

```cmake
set(CMAKE_C_STANDARD_REQUIRED ON)
```

## CMAKE_C_STANDARD
指定c标准版本
```cmake
#指定为c11标准版本
set(CMAKE_C_STANDARD 11)
```

## CMAKE_CXX_STANDARD_REQUIRED
设置必须指定C++标准版本

```cmake
set(CMAKE_CXX_STANDARD_REQUIRED ON)
```

## CMAKE_CXX_STANDARD
指定c++标准版本

默认为OFF

```cmake
# 指定为C++11 版本
set(CMAKE_CXX_STANDARD 11)
```
