## scrapy命令

```bash
scrapy startproject <项目名>					    #创建一个新项目。
scrapy genspider <爬虫文件名> <域名>			     #新建爬虫文件。
scrapy runspider <爬虫文件>                       #运行一个爬虫文件，不需要创建项目。
scrapy crawl <spidername>                       #运行一个爬虫项目，必须要创建项目。
scrapy list										#列出项目中所有爬虫文件。
scrapy view <url地址>							   #从浏览器中打开 url 地址。
csrapy shell <url地址>						   #命令行交互模式。
scrapy settings 								#查看当前项目的配置信息。
```
