## 变化

C++11 首次引入 `[[attribute]]` 语法，比如：

```
cpp


复制编辑
[[nodiscard]] int foo();
```

但在 C++11/14 中，**如果你使用了一个当前版本不支持的属性，编译器会报错**：

```
cpp


复制编辑
[[unknown_attribute]] int x; // ❌ error: attribute not recognized
```

这让**向后兼容**和**跨版本编程**变得非常困难。

C++17 明确规定： 如果属性在当前语言版本中未定义，但其语法是有效的标准属性（即使用 `[[...]]` 语法），**编译器必须忽略它**，而不能报错。
