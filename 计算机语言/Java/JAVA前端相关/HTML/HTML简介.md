## HTML
HTML全程为超文本标记语言(Hyper Text Markup Language)，HTML通过标签来标记显示网页中的各个部分。网页本身就是一种文本格式的文件。


## HTML标签
> 格式

    <标签名 [标签属性1="value1"] [标签属性2="value2"] ...> 封装的数据 <标签名>
    //或
    <标签名 [标签属性1="value1"] [标签属性2="value2"] ... />

* 标签名对大小写不敏感    

* 属性值必须用""扩起来

## HTML注释
> 多行

    <!--
        
        注释内容
    
    -->

> 单行
    
    <!-- 注释内容 -->
