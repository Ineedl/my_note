## 前置依赖log4j

* 注意log4j高危BUG，请使用最新的修复了该BUG的版本


    <!-- https://mvnrepository.com/artifact/org.apache.logging.log4j/log4j-core -->
    <dependency>
        <groupId>org.apache.logging.log4j</groupId>
        <artifactId>log4j-core</artifactId>
        <version>版本</version>
    </dependency>
